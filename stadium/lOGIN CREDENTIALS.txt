HOW TO USE THE STADIUM TICKET BOOKING SYSTEM


1. UNZIP THE SMS FOLDER
2. OPEN THE SMS FOLDER TO A CODE EDITOR
3. MOVE TO THE DIRECTORY CALLED STADIUM BY TYPING THE COMMAND "cd stadium"
4. RUN THE SERVER USING THE COMMAND "python manage. py runserver"

YOU ARE DONE!
